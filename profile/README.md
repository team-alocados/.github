## Alocados(알로카도스)에 오신 것을 환영합니다. 👋

**알로카도스는 블록체인 기술이 바꿔나갈 Web 3.0 시대를 꿈꾸는 크립토 스타트업입니다.**
알로카도스 팀은 카카오, 카카오모빌리티, 알리안츠 등에서의 다양한 경험을 가진 팀원들로 구성되었습니다.
